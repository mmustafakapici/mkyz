Data Processing¶

This module provides tools for preparing data for machine learning tasks.

### Functions

mkyz.data_processing.prepare_data(filepath, target_column=None, numerical_columns=None, categorical_columns=None, test_size=0.2, random_state=42, binary_threshold=2, low_cardinality_threshold=10, drop_columns=None, outlier_strategy='remove', categorical_transform_method='onehot')¶

Veriyi hazırlamak için eksik değerlerin doldurulması, aykırı değerlerin işlenmesi, kategorik değişkenlerin dönüştürülmesi ve veri setinin bölünmesi işlemlerini gerçekleştirir.

Parameters:

  • filepath (str) – Veri setinin CSV dosya yolu.

  • target_column (str, optional) – Hedef sütun adı. Sağlanmazsa, son sütun kullanılır. Defaults to None.

  • numerical_columns (list, optional) – Sayısal sütunlar. Defaults to None.

  • categorical_columns (list, optional) – Kategorik sütunlar. Defaults to None.

  • test_size (float, optional) – Test veri oranı. Defaults to 0.2.

  • random_state (int, optional) – Rastgelelik kontrolü için seed. Defaults to 42.

  • binary_threshold (int, optional) – Sütunun binary olarak kabul edilebilmesi için maksimum benzersiz değer sayısı. Defaults to 2.

  • low_cardinality_threshold (int, optional) – Düşük kardinaliteli sütunlar için maksimum benzersiz değer sayısı. Defaults to 10.

  • drop_columns (list, optional) – Veri setinden çıkarılacak sütunlar. Defaults to None.

  • outlier_strategy (str, optional) – Aykırı değer işleme stratejisi (‘remove’, ‘cap’). Defaults to ‘remove’.

  • categorical_transform_method (str, optional) – Kategorik dönüşüm yöntemi (‘onehot’, ‘frequency’, ‘target’). Defaults to ‘onehot’.

Returns:

(X_train, X_test, y_train, y_test, df, target_column, numerical_columns, categorical_columns)

Return type:

tuple

Usage:
>>> data = prepare_data(
        'data.csv', target_column='target', drop_columns=['ID'],
        outlier_strategy='cap', categorical_transform_method='frequency')

Prepares the dataset for training by handling missing values, encoding categorical variables, and splitting the data into features and target labels.

Usage:

from mkyz import data_processing as dp
data = dp.prepare_data('winequality-red.csv', target_column='quality', task='classification')

Arguments: - data: A path to the dataset (CSV file) or a pandas DataFrame. - target_column: The target variable (classification or regression task). - task: The type of task (‘classification’ or ‘regression’).

Returns: - A tuple of (X, y) where X is the feature matrix and y is the target variable.
