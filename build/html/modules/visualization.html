Visualizes the data using the specified plot type.

Parameters:

  • data (pandas.DataFrame or array-like) – Data to visualize.

  • plot_type (str, optional) – The type of plot to generate (e.g., ‘scatter’, ‘bar’, ‘heatmap’, ‘corr’). Defaults to ‘scatter’.

  • **plot_params – Additional parameters for the plot (e.g., title, labels, color).

Returns:

Displays the plot.

Return type:

None

Usage:
>>> from mkyz import visualization as viz
>>> viz.visualize(data=data, plot_type='corr')

Generates different types of plots to help visualize your data or model results.

Usage:

from mkyz import visualization as viz
viz.visualize(data=data, plot_type='corr', title='Correlation Matrix')

Arguments: - data: The dataset to visualize. - plot_type: The type of plot (e.g., ‘scatter’, ‘corr’). - **plot_params: Additional parameters for the plot such as title, labels, etc.

Returns: - Displays the generated plot.
