Introduction
============

Welcome to the **mkyz** library documentation!

**mkyz** is a comprehensive machine learning library that simplifies data processing, model training, evaluation, and visualization.

Key Features:
- **Data Preparation**: Efficiently process your datasets, handling missing values, encoding categorical variables, and splitting data for training.
- **Model Training**: Supports classification, regression, clustering, and other machine learning tasks.
- **Evaluation**: Easy-to-use model evaluation tools with detailed metrics.
- **Visualization**: Provides a variety of graphical representations, from scatter plots to correlation matrices.
